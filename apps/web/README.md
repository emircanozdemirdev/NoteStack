# @smartnotes/web

Frontend web app for SmartNotes.

## Stack (planned)

- Next.js (App Router) + TypeScript
- Tailwind CSS
- React Query or SWR (TBD)
- NextAuth.js or custom JWT-based auth

## Status

The real Next.js application will be scaffolded in **Phase 3**.
