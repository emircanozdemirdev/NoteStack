# @smartnotes/api

Backend API for SmartNotes.

## Stack (planned)

- Node.js 20 LTS
- NestJS (TypeScript)
- Prisma ORM
- PostgreSQL
- JWT-based authentication

## Status

This is a **placeholder** workspace created in Phase 1.
The real NestJS application will be scaffolded in **Phase 2**.

## Scripts

All scripts currently print a placeholder message. They exist so that
pnpm workspace commands (`pnpm -r run dev`, `pnpm -r run build`, ...) can
discover this workspace correctly.
